# vue-shopping-trolley

WIP : 使用 Vue+Vite+TypeScript+Less+Pinia +Naive UI 构建的前端简易购物车

## Project Setup

```sh
pnpm install
```

### Compile and Hot-Reload for Development

```sh
pnpm dev
```

### Type-Check, Compile and Minify for Production

```sh
pnpm build
```
